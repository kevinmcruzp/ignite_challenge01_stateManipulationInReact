Installation
============
```
#Clone the repository
  $ git clone git@github.com:KevinMCruzP/challengeReact.git
```

```
#Install the dependencies
  $ npm install

or

  $ yarn
```

```
#Run the app in http://localhost:8080/
  $ npm run dev

or

  $ yarn dev
```

```
#Automatic mode to evaluate the challenge
  $ npm run test

or

  $ yarn test
```
